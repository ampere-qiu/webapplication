## 3. PDF レポートに含める内容

レポートでは、次の項目を必ず含めること。

- 外部仕様
- 内部仕様
- Socket.IO イベント一覧表
- React コンポーネント構成

- 工夫した点
- AI 使用に関する記述

!!! tip "レポート作成のコツ"
    まずは簡単な構成で書き、後から図や表を追加すると整理しやすい。  
    画面構成やイベントの流れを図にすると、説明が分かりやすくなる。

## 4. 提出前に確認すること

!!! warning "重要"
    アプリ画面の上部に、名前と学籍番号を表示すること。  
    これが書かれていない場合は、提出物として確認しづらく、評価対象にならないことがある。

- ゲームが動作するか
- ルールが説明できるか
- ソースコードの構成が整理されているか
- レポートに必要な項目が揃っているか
- 最終提出前に、画面表示とレポート内容を確認しているか
